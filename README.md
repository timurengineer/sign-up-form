# Sign Up Form

![screenshot](./screenshot.png)

## Prerequisites

- Node.js 10.15.1
- MongoDB 4.0.3

## Quick Start
```
git clone https://github.com/timurengineer/sign-up-form.git
cd sign-up-form
npm install
```
Start API server
```
npm run api-dev
```
Start Front End Application
```
npm start
open localhost:3000 in your browser
```
